# Personal Portfolio Website

This is a simple personal portfolio website built for a lab exercise. 

## About the Project
This project uses only pure HTML (no CSS or JavaScript). It includes three main pages:
1. Home Page (`index.html`)
2. Projects Page (`projects.html`)
3. Contact Page (`contact.html`)

## Purpose
The goal of this project is to practice creating HTML pages, setting up a GitHub repository, making branches, and resolving Git merge conflicts.
